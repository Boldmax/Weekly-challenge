My Favourite Git Command

What is a git command?

Git is a fast, scalable, distributed revision control system with
an unusually rich command set that provides both high-level
operations and full access to internals.

My favourite git command is "git commit".

What does git commit does?

Git commit command is the main function for saving changes
to local repository safely. Commit takes a snapshot of the
project and treats it as a version.

Git commit flag, M.

Git commit is normally used with an m flag. The m flag is used
for connecting a commit message to your commit for example
git commit -m "your message"

A simple example of how git commit is used.

Let's take a look of how git commit is used.

  1. Create a directory named "Commit_folder" using the "mkdir" command.
  2. Create a new file, "commit.txt" using the "touch commit.txt" command.
  3. Before commiting a change, we need to add the change to the local
    repo by using "git add ." command.
  4. And finally we can commit our changes by using the
    "git commit -m 'I am trying to create a commit example' command.
